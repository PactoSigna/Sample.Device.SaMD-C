# User Need: Clinician Review of Patient Data

## Description

As a clinician reviewing patient ECG data, I need to quickly understand the detection results and confidence levels so that I can make informed treatment decisions.

## Context

Clinicians review many patient records daily. They need:

- Clear visual presentation of detection results
- Confidence indicators to understand reliability
- Ability to drill into raw ECG data when needed
- Audit trail of when data was recorded and analyzed

## Success Criteria

1. Detection results displayed prominently
2. Confidence score clearly visible and understandable
3. Raw ECG data accessible for review
4. Timestamps and patient context available

## Stakeholders

- **Primary**: Cardiologists, primary care physicians
- **Secondary**: Nurses and clinical staff
- **Tertiary**: Hospital IT administrators
