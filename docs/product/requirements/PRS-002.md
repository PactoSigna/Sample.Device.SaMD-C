# Product Requirement: Real-time Monitoring Dashboard

**Derives from:** [UN-002](../user-needs/UN-002.md)

## Description

The system shall provide a real-time dashboard displaying ECG data, detection results, and patient status for clinician review.

## Rationale

Clinicians need immediate access to patient data to make timely treatment decisions. A real-time dashboard enables efficient workflow and reduces time-to-decision.

## Acceptance Criteria

- Dashboard displays detection results within 5 seconds of analysis completion
- Multiple patients viewable simultaneously
- Detection history accessible for trend analysis
