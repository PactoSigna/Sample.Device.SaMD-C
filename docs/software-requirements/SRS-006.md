# Software Requirement: Detection Result Persistence

**Derives from:** [PRS-002](../product-requirements/PRS-002.md)

## Description

The software shall persist all detection results to the database with associated metadata for historical review.

## Technical Specification

Persisted data:

- Detection result (AFib detected: yes/no)
- Confidence score
- Timestamp of analysis
- ECG segment reference
- Signal quality at time of analysis
- Algorithm version

## Acceptance Criteria

- Results persisted within 1 second of analysis
- Results retrievable for 7+ years per retention policy
- Historical results queryable by patient, date range
