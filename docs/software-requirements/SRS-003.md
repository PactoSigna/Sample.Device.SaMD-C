# Software Requirement: Confidence Score Display

**Derives from:** [PRS-002](../product-requirements/PRS-002.md), [PRS-003](../product-requirements/PRS-003.md)

## Description

The software shall display detection confidence as a percentage with color-coded visual indicator on the clinician dashboard.

## Technical Specification

- Display: Numerical percentage (0-100%) and progress bar
- Color coding:
  - Green (#4CAF50): â‰¥90% confidence
  - Yellow (#FFC107): 70-89% confidence
  - Red (#F44336): <70% confidence
- Tooltip: Explains confidence calculation methodology

## Acceptance Criteria

- Confidence visible within 1 second of detection result
- Color matches confidence threshold exactly
- Tooltip accessible on hover/tap
