# Software Requirement: Audit Log Generation

**Derives from:** [PRS-004](../product-requirements/PRS-004.md)

## Description

The software shall generate immutable audit log entries for all data creation, modification, deletion, and access events.

## Technical Specification

Audit entry fields:

- Timestamp (ISO 8601, UTC)
- User ID
- Action type (create, read, update, delete)
- Resource type and ID
- Previous value (for updates)
- New value (for updates)
- IP address
- Session ID

## Acceptance Criteria

- All covered events generate audit entry
- Entries written within 100ms of event
- No mechanism to delete or modify entries
- Entries queryable by time range, user, resource
