# Test Case: Confidence Display Verification

**Verifies:** [SRS-003](../software-requirements/SRS-003.md)
**Validates:** [UN-002](../user-needs/UN-002.md)

## Objective

Verify that confidence scores are displayed correctly with appropriate color coding.

## Procedure

1. Generate detection result with 95% confidence
   - Expected: Green indicator, "95%" displayed
2. Generate detection result with 75% confidence
   - Expected: Yellow indicator, "75%" displayed
3. Generate detection result with 60% confidence
   - Expected: Red indicator, "60%" displayed
4. Hover over confidence indicator
   - Expected: Tooltip explains confidence methodology

## Pass Criteria

- Percentage displays correctly (+/-1%)
- Color matches threshold:
  - Green: >=90%
  - Yellow: 70-89%
  - Red: <70%
- Tooltip accessible and accurate

## Expected Results

| Confidence | Expected Color   | Expected Text |
| ---------- | ---------------- | ------------- |
| 95%        | Green (#4CAF50)  | "95%"         |
| 90%        | Green (#4CAF50)  | "90%"         |
| 89%        | Yellow (#FFC107) | "89%"         |
| 70%        | Yellow (#FFC107) | "70%"         |
| 69%        | Red (#F44336)    | "69%"         |
| 50%        | Red (#F44336)    | "50%"         |
